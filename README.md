# Taniti Island Tourism Website 

A responsive, single-page tourism website designed for the fictional island destination **Taniti**.  
This project showcases a contemporary, user-centered approach to web design using **React** and **Tailwind CSS**, with a focus on clear navigation, visual consistency, and an inviting travel experience.

---

## Project Overview

The Taniti Tourism Website provides visitors with essential information to plan a trip to the island, including:

- Island experiences and attractions  
- Dining options and nightlife  
- Accommodation choices  
- Transportation and arrival information
- Essential travel information 

The site is designed to feel **polished, approachable, and easy to navigate**, encouraging users to explore and plan their visit.

---

## Features

- Single-page layout with smooth anchor navigation  
- Reusable React components for cards and section headers  
- Responsive design optimized for desktop, tablet, and mobile  
- Subtle hover animations  
- Consistent visual hierarchy and typography  
- Accessible, scannable content structure  

---

## Technologies Used

- **React** – component-based UI development  
- **Vite** – fast development and build tooling  
- **Tailwind CSS** – utility-first styling and responsive design  
- **JavaScript XML (JSX)**  
- **HTML5 & CSS3**
